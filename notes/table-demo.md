# テーブル編集デモ

セルをクリックすると編集できます。区切り行の `:` で列の揃え(左/中央/右)が反映されます。

| 部品名 | 数量 | 状態 |
| --- | ---: | :---: |
| 抵抗 | 100 | 在庫あり |
| コンデンサ | 50 | 発注中 |
| IC | 20 | 入荷待ち |

## セル内の WikiLink

セル内の `[[...]]` はクリックでそのノートを開けます(リンク以外の余白をクリックすると編集に入ります)。

| ノート | 関連リンク |
| --- | --- |
| 概要 | [[welcome]] |
| 企画 | [[project-ideas|アイデア一覧]] |
