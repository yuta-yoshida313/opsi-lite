# Welcome to Opsi-Lite

これはサンプルノートです。`opsi-lite notes/welcome.md` で起動して動作確認できます。

## WikiLink のテスト

別のノートへのリンク: [[project-ideas]] や [[daily/2026-06-10|今日のメモ]] を試してください。
カーソルを `[[` の上に置いて **F12** でジャンプできます。

## タスクのテスト

- [ ] Ctrl+Enter でこの行をトグル
- [x] 完了済みのタスク
- [ ] プレビュー側のチェックボックスもクリックしてみる

## コードブロック

```rust
fn main() {
    println!("hello opsi-lite");
}
```

> 引用ブロックのサンプル。インライン `code` や **強調**、*斜体* も色付けされます。

## テーブルのテスト

プレビュー側のセルをクリックすると、その場で編集できます（Enterで次のセルへ）。

| 部品名 | 数量 | 状態 |
| --- | ---: | --- |

| コンデンサ | 50 | 発注中 |
